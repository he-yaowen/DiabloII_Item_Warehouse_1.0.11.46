# Item Warehouse for Diablo II LoD version 1.0.11.46

This repository contains item files (.d2i) for Diablo II LoD version 1.0.11.46
(so-called 1.10)

## How to Use

Use any editor (e.g. [ATMA][1]) to import items (.d2i) into your saved
character files.
